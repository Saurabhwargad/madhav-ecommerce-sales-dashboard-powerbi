# Madhav E-Commerce Sales Dashboard - Power BI

## Project Overview
This Power BI dashboard analyzes sales and profit performance of Madhav E-commerce business. The objective is to identify revenue drivers, profitable product categories, customer trends, and regional performance.

## Key Metrics
- Total Sales Amount: 438K
- Total Profit: 37K
- Total Quantity Sold: 5615
- Average Order Value (AOV): 121K

## Dashboard Insights

### Sales Analysis
- Maharashtra contributes highest sales.
- Clothing category dominates quantity share (63%).
- COD is the most used payment method (44%).

### Profit Trends
- Profit fluctuates across months.
- Highest profit observed in December.
- Printers and Bookcases generate maximum profit in sub-category.

## Features
- State-wise sales analysis
- Monthly profit trend
- Category & sub-category performance
- Payment mode distribution
- Customer-level revenue insights

## Tools Used
- Power BI
- DAX Measures
- Data Modeling
- Data Cleaning & Transformation

## Objective
Transform transactional data into actionable insights to support strategic business decisions.
